Detailed Description

Implementation of tuner functionality.

This file includes the whole functionality to adjust the PI tuner network on tuner enabled boards. The communication pins for the tuner network are defined in platform.h. The tuner network consists of 3 variable capacitors: Cin, Clen and Cout and looks like this:

.--- L ---.
| |
in ----.----.----- L --.--C_len--.--.-----.-----.----- out
| | | | |
C_in L C_out L R
| | | | |
