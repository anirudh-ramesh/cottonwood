void timerStart_us ( u16  us)

Start a timer which generates an interrupt after us microseconds. Use TIMER_IS_DONE() to poll expiration. No interrupt is being used, make sure no ISR clears the interrupt bit.

Parameters
us: microseconds until expiration.
